<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| network -> FastAPI | The dev server listens on a socket; binding beyond localhost would expose it |
| process -> disk (.patchpilot_memory/) | Ingested content is persisted to disk and could be committed by accident |
| config -> OpenAI | cognee_config carries the model/provider that governs spend |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-01-04 | Information Disclosure / Elevation | uvicorn bind address | low | mitigate | Run `uvicorn --host 127.0.0.1 --workers 1`; never `0.0.0.0` in Phase 1; documented in main.py docstring and verify command |
| T-01-02 | Information Disclosure | `.patchpilot_memory/` on disk | medium | mitigate | `.patchpilot_memory/` is gitignored (Plan 01); health fixture is `forget()`-cleaned each call so throwaway data does not accumulate |
| T-01-01 | Information Disclosure | env/secret load in cognee_config | high | mitigate | cognee_config reads the key from the gitignored `.env` via `load_dotenv()`; no secret is hardcoded in any `backend/*.py`; verified by grep for `sk-` in tracked files |
| T-01-03 | Denial of Service (self, cost) | per-call cognify in /health/cognee | medium | mitigate | Fixture is a single sentence on `gpt-4o-mini`; endpoint is an exit-gate check, not a hot path; $10 cap active from Plan 01 |
</threat_model>
